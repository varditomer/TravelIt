Welcome to TravelIt

Your ultimate travel companion app built with Angular and deployed on Firebase. TravelIt simplifies the trip planning process and enhances your travel experiences, allowing you to explore the world with ease.

Key Features:

  • Trip Planning Made Easy

    Effortlessly create and manage your travel itineraries. TravelIt displays personalized travel plans based on destination, travel dates, and notes.

  • Explore Destinations

    Discover countries and their flags by input country name.

Installation

  1. Clone the repository: git clone https://github.com/your-username/TravelIt.git
  2. Install dependencies: npm install
  3. Start the development server: ng serve
  4. Open your browser and navigate to http://localhost:4200

Live Demo

Check out the live demo of TravelIt https://travelit-tomer.firebaseapp.com/.
